# Documents Folder

## Struktur Folder
```
/public/documents/
├── profil-desa-2024.pdf      # File PDF buku profil desa utama
└── README.md                  # File ini
```

## Petunjuk Upload PDF

1. **File PDF Buku Profil**: 
   - Letakkan file PDF buku profil dengan nama: `profil-desa-2024.pdf`
   - Pastikan ukuran file tidak lebih dari 10MB
   - Format harus PDF yang bisa dibuka di browser

2. **Akses File**:
   - File akan dapat diakses di: `/documents/profil-desa-2024.pdf`
   - Sistem akan otomatis mendeteksi file dan menampilkan informasinya

3. **Cover Image** (Opsional):
   - Jika ada cover image, letakkan di `/public/images/book-cover.png`
   - Format: PNG (atau JPG), ukuran maksimal 2MB
   - Dimensi yang disarankan: 600x800px (rasio buku)
